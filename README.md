**Maven**

1. Download dự án về máy bằng link liên kết này:[Chat App](https://github.com/Nqvinh7603/CT240-Chat-App)

3. Cài đặt JDK 20 và apache maven
4. Cài đặt MySql workbench và tạo cơ sở dữ liệu có tên là **chatapp**
